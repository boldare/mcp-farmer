# Project Overview

This is an MCP (Model Context Protocol) server.

## MCP SDK

This project uses the official TypeScript SDK for MCP: `@modelcontextprotocol/sdk`.
It provides the core functionality for creating MCP servers, defining tools, resources, and prompts.

## Project Structure

- `src/` - Source code directory
  - `server.ts` - Main server logic and tool registration
  - `tools/` - Directory for tool definitions. **New tools should be added here.**
